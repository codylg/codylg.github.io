Very Usable Icon Library

The Very Usable Icon Library is a free to use component library for Figma with over 550 icons. Some features of this library are:

  • Tagged/alternative icon names for easy searching
  • Categorised groups for browsing
  • Flattened, uniform layers so colour overrides are preserved when switching icon instances
  • 2 size variants (18px and 12px) but also scalable
  • Sharp rendering on low density displays
  • Optically balanced (to a degree) to create more uniform weight
Open in Figma

iOS'86

iOS'86 was a personal project that developed into one of the most expansive iPhone themes of the time. Based on the concept designed by Anton Repponen, iOS'86 is a fully functional Winterboard theme for iPhone.

Including thousands of hand-pixeled monochrome graphics along with a port of the original Chicago font; iOS'86 takes your device back to another era. Available on iOS 4, 5, and 6.
